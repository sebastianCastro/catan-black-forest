Forest Tiles ? Enter the tile numbers that should be forest terrain. These tiles will remain fixed as forest regardless of randomization. Tile numbers start from 1 at the top and increase row by row.
Desert Tiles ? Enter the tile numbers that should be desert terrain. Desert tiles don't produce resources. The robber starts on a desert tile.
Port Tiles ? Enter the tile numbers for port locations. Ports allow for more favorable trading ratios when settlements or cities are adjacent to them.
Port Tile Rotations ? Define the rotation of each port in degrees (0-360). Format as a JSON object with tile numbers as keys and rotation angles as values. This affects which edge of the hexagon the port faces.
Fog Terrain Distribution ? Set how many of each terrain type should appear in the "fog" (unexplored) areas of the map. These will be randomly assigned to non-forest, non-desert, and non-port tiles.
Number Distributions ? Control how many "red" numbers (6 and 8) appear on the map. These are the most valuable numbers since they have the highest probability of being rolled.
